# 🎨 Premium Bio & Caption Generator Bot

A production-ready Telegram bot that generates viral bios, captions, and statuses for Telegram, Instagram, TikTok, and other social media platforms.

## 🌟 Features

### Bio Generators
- 📱 Telegram Bio Generator
- 📸 Instagram Bio Generator
- 🎥 TikTok Bio Generator
- 🌑 Dark Vibe Bios
- 💎 Luxury/Rich Vibes
- ❤️ Love & Couple Bios
- 😢 Emotional/Sad Bios
- 👿 Mafia Style Bios
- 📚 Student Bios
- 💼 Business Bios
- ⚽ Football Fan Bios
- 🏎️ Car Lover Bios

### Additional Features
- 🎭 Username Generator
- ✨ Stylish Text Generator
- 📹 Reel Caption Generator
- 📸 Story Caption Generator
- 💬 Emoji-Rich Captions

### Referral System
- 👥 Unique referral links for each user
- 🎯 Unlock premium content at milestones:
  - 2 referrals: Dark Vibe, Luxury Vibes
  - 5 referrals: Emotional, Mafia Bios
  - 10 referrals: Silver Status
  - 25 referrals: Gold Status
  - 50+ referrals: Legend Status

### Ranking System
- 🌱 NOVICE (0 referrals)
- 🥉 BRONZE (2 referrals)
- ⭐ SILVER (5 referrals)
- 🏆 GOLD (10 referrals)
- 💎 DIAMOND (25 referrals)
- 🔱 LEGEND (50+ referrals)

### Gamification
- 🎁 Daily Bonus System
- 🔥 Streak Tracking
- 🏆 Leaderboard with Top Inviters
- 💰 Reward System
- 📊 User Statistics

### Admin Panel
- 👥 View Total Users
- 📊 View Bot Statistics
- 📢 Broadcast Messages
- 👑 Give/Remove Premium
- ⛔ Ban/Unban Users
- 📤 Export Users to CSV
- 🏆 View Top Inviters
